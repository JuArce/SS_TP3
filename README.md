# Generador

Parámetros:
* `n`: cantidad de partículas.
* `Lx`: largo del sistema en x (0.12).
* `Ly`: largo del sistema en y (0.09).
* `staticNFile`: nombre del archivo estático de salida
* `dynamicNFile` nombre archivo dinámico de salida
